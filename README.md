# PocketFriendly
(PROJECT) A smart budget tracker that provides important details such as spending insights. 

With this budget tracker, the user should be able to:
1. Gain control over their finances.
2. Practice better saving habits.
3. Increase their financial awareness and planning.

Key Features of this application include:
1. Dashboard that is easy to use and navigate through, providing useful information such as transaction history and spending summary.
2. Smart insights to notify the user about their spending habits and alert them incase of overspending.
3. Pie Chart to visalize all the user's expenses.
4. Search, edit and delete transactions

the application is easy to use. Simply input your transaction amount, its category (such as Food, Business, etc.) and select whether it was an expense or income. The search bar, edit and delete button are also very accessible.
